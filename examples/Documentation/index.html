Techwind - Laravel 12 Multipurpose App, Saas & Software Landing & Admin Dashboard Template

Introduction

Techwind is a Powerful Laravel 12 Multipurpose App, Saas & Software Landing & Admin Dashboard Template. It is a beautifully handcrafted, pixel-perfect landing page template based on Laravel v12.14.1 PHP v8.4.4 & Tailwind CSS v4.1.6. It is an excellent template for startups, cloud hosting, business, corporate, minimal portfolio single products (Smartwatch), Saas, Social media marketing, Digital marketing/agency, Email Templates, Marketing, Agency, Careers and Jobs, Helpcenter, and much more. It is 100% responsive and looks stunning on all types of screens and devices. Users will love your site because it gives them a unique user experience (UX), and a clean, modern & beautiful design. They are also easy to customize and combine with other components.

We have added Dark version and RTL vesion..

To spin up the Laravel server, follow the below mentioned steps in a separate terminal:

Command Description
composer i This would install all the required dependencies in the vender folder.
cp .env.example .env This will copy .env.example as .env file.
php artisan key:generate This will generate a unique key and write it in your .env file
php artisan migrate This will executes all pending migrations to database.
php artisan serve Runs the project locally, starts the development server on http://127.0.0.1:8000.

Contact Form Configuration:

  • For .env File

    Update the following lines in the .env file to configure email sending:

MAIL_MAILER=smtp
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
MAIL_USERNAME=your-email@gmail.com # Replace with your email
MAIL_PASSWORD="" # Replace with the app-specific password
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS="your-email@gmail.com" # Replace with your email
MAIL_FROM_NAME="Techwind" # Replace with your project name or desired sender name

Key Points:

  • 1. MAIL_USERNAME and MAIL_FROM_ADDRESS:

  • Ensure both are the same and match the email address you’re using to send the emails.

  • 2. MAIL_PASSWORD:

  • This should be the App Password generated for your email account, not your actual email password.

  • For Gmail, app passwords can be generated from your Google Account if 2-Step Verification is enabled.

  • Refer to this Google support link to create an app password.

  • 3. MAIL_FROM_NAME:

  • "Techwind" is fine as a default project name. You can customize it to represent your project’s branding.

Testing Instructions

  • You can test your email configuration in the following ways:

  • 1. Using Tinker:

  • Run the following commands in the terminal:

  • php artisan tinker

  • Then, in the Tinker shell, execute this code:

    • Mail::raw('Test email', function ($message) {
    $message->to('recipient@example.com')->subject('Testing Email');
    });

  • Replace recipient@example.com with the email address where you want to send the test email.

  • If the email configuration is correct, you should receive the test email.

  • 2. Using the Frontend Contact Form:

  • If you have a contact form on the frontend, fill it out and submit it.

  • Check both the sender’s and recipient’s inboxes to confirm successful delivery.

Common Errors Section

  • To assist with troubleshooting, here are some common issues and solutions:

  • 1. Incorrect Credentials:

  • Double-check the email, password, and SMTP settings in the .env file.

  • 2. 2-Step Verification Disabled:

  • Ensure 2-Step Verification is enabled on your Google account to use app passwords.

  • 3. App Password Not Generated:

  • Create an app-specific password for your Gmail account to ensure compatibility.

  • 4. Gmail Blocking Less Secure Apps:

  • Use app-specific passwords instead of your regular email password to bypass this restriction.
